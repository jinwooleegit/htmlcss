HTML 완전 정복 튜토리얼

HTML 기초부터 고급 기술까지 체계적으로 배워보세요

HTML이란 무엇인가요?

HTML = HyperText Markup Language

HyperText

링크로 연결된 텍스트

Markup

구조를 표시하는 방법

Language

컴퓨터가 이해하는 언어

쉬운 비유로 이해하기

웹페이지를 집이라고 생각해보세요:

  • HTML = 집의 뼈대 (기둥, 벽, 방)
  • CSS = 집의 인테리어 (색깔, 가구, 장식)
  • JavaScript = 집의 전자제품 (조명, 에어컨, TV)

왜 HTML을 배워야 할까요?

모든 웹사이트의 기초

구글, 페이스북, 유튜브를 포함한 모든 웹사이트가 HTML을 기반으로 구축됩니다. 웹 개발의 첫걸음이자 가장 중요한 기초 기술입니다.

배우기 쉬운 언어

프로그래밍 경험이 없어도 쉽게 배울 수 있습니다. 직관적인 태그 구조로 누구나 빠르게 익힐 수 있는 마크업 언어입니다.

다양한 분야에서 활용

웹 개발자뿐만 아니라 디자이너, 마케터, 콘텐츠 제작자 등 다양한 분야에서 필요로 하는 핵심 기술입니다.

즉시 확인 가능한 결과

코드를 작성하면 브라우저에서 바로 결과를 확인할 수 있어 학습 효과가 뛰어나고 성취감을 빠르게 느낄 수 있습니다.

HTML 문법의 기초

HTML의 기본 구조: 태그(Tag)

< 태그이름 > 내용 </ 태그이름 >
여는 태그: <태그이름>
내용: 실제 표시될 텍스트
닫는 태그: </태그이름>

HTML 문법 규칙

대소문자 구분 안함

<H1>과 <h1>은 동일하게 처리됩니다

태그는 중첩 가능

<div><p>내용</p></div>

빈 태그 존재

<br>, <img> 등은 닫는 태그가 없습니다

들여쓰기 권장

가독성을 위해 적절한 들여쓰기를 사용하세요

첫 번째 실습

아래 코드를 따라 써보세요:

<h1>안녕하세요!</h1>
<p>이것은 내 첫 번째 HTML 태그입니다.</p>
결과:

HTML 문서의 기본 구조

모든 HTML 문서는 이 구조를 가져야 해요

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>페이지 제목</title>
</head>
<body>
    <h1>여기에 내용을 작성하세요</h1>
    <p>이것은 웹페이지의 본문입니다.</p>
</body>
</html>

제목 태그 (Heading Tags)

<h4>네 번째 제목</h4>
<h5>다섯 번째 제목</h5>

줄바꿈 태그

줄을 바꿀 때 사용하는 태그입니다. 닫는 태그가 없어요!

<p>첫 번째 줄<br>두 번째 줄<br>세 번째 줄</p>

텍스트 강조 태그

텍스트를 강조하거나 꾸밀 때 사용하는 태그들입니다.

<p>
    <strong>굵은 글씨</strong><br>
    <em>기울어진 글씨</em><br>
    <u>밑줄 글씨</u><br>
    <s>취소선 글씨</s>
</p>

텍스트 서식 태그

텍스트를 더 의미있고 아름답게 꾸미는 다양한 태그들을 배워보겠습니다.

의미적 강조 태그

단순히 모양만 바꾸는 것이 아니라, 의미를 전달하는 태그들입니다.

<p>
    <strong>매우 중요한 내용</strong> (굵게 + 의미적 강조)<br>
    <em>강조하고 싶은 내용</em> (기울임 + 의미적 강조)<br>
    <mark>하이라이트된 텍스트</mark> (형광펜 효과)<br>
    <small>작은 글씨 (저작권, 부가정보)</small>
</p>

매우 중요한 내용 (굵게 + 의미적 강조)
강조하고 싶은 내용 (기울임 + 의미적 강조)
하이라이트된 텍스트 (형광펜 효과)
작은 글씨 (저작권, 부가정보)

편집 관련 태그

텍스트의 변경사항을 표시하는 태그들입니다.

<p>
    원래 가격: <del>10,000원</del><br>
    할인 가격: <ins>8,000원</ins><br>
    <s>품절된 상품</s> → 재입고 완료!
</p>

원래 가격: 10,000원
할인 가격: 8,000원
품절된 상품 → 재입고 완료!

특수 텍스트 태그

특별한 용도로 사용되는 텍스트 태그들입니다.

<p>
    화학 공식: H<sub>2</sub>O (아래첨자)<br>
    수학 공식: E=mc<sup>2</sup> (위첨자)<br>
    코드: <code>console.log('Hello')</code><br>
    키보드: <kbd>Ctrl + C</kbd> (키보드 입력)<br>
    변수: <var>x + y = z</var> (수학 변수)
</p>

화학 공식: H2O (아래첨자)
수학 공식: E=mc2 (위첨자)
코드: console.log('Hello')
키보드: Ctrl + C (키보드 입력)
변수: x + y = z (수학 변수)

링크와 이미지

링크 만들기

다른 페이지나 사이트로 이동할 수 있는 링크를 만드는 방법입니다.

<!-- 다른 웹사이트로 링크 -->
<a href="https://www.google.com">구글로 이동</a>

<!-- 새 창에서 열기 -->
<a href="https://www.naver.com" target="_blank">네이버 (새 창)</a>

<!-- 같은 사이트 내 다른 페이지 -->
<a href="css-tutorial.html">CSS 튜토리얼</a>

<!-- 이메일 링크 -->
<a href="mailto:someone@example.com">이메일 보내기</a>

<!-- 전화번호 링크 -->
<a href="tel:010-1234-5678">전화걸기</a>

구글로 이동
네이버 (새 창)
CSS 튜토리얼
이메일 보내기
전화걸기

링크 사용 팁

  • target="_blank": 새 창에서 열기
  • href="#": 페이지 맨 위로 이동
  • href="#section1": 같은 페이지 내 특정 위치로 이동

이미지 추가하기

웹페이지에 이미지를 표시하는 방법입니다.

<!-- 기본 이미지 -->
<img src="photo.jpg" alt="사진 설명">

<!-- 크기 지정 -->
<img src="photo.jpg" alt="사진" width="300" height="200">

<!-- 링크가 있는 이미지 -->
<a href="https://www.example.com">
    <img src="logo.jpg" alt="로고">
</a>

이미지가 여기에 표시됩니다

실제로는 이미지 파일이 필요합니다

중요한 속성들

  • src: 이미지 파일의 경로 (필수)
  • alt: 이미지 설명 텍스트 (접근성을 위해 필수)
  • width, height: 이미지 크기 (픽셀 단위)

목록 만들기

순서 없는 목록 (Unordered List)

순서가 중요하지 않은 항목들을 나열할 때 사용합니다.

<ul>
    <li>사과</li>
    <li>바나나</li>
    <li>오렌지</li>
</ul>
  • 사과
  • 바나나
  • 오렌지

순서 있는 목록 (Ordered List)

순서가 중요한 항목들을 나열할 때 사용합니다.

<ol>
    <li>물을 끓인다</li>
    <li>라면을 넣는다</li>
    <li>3분 기다린다</li>
    <li>완성!</li>
</ol>
  1. 물을 끓인다
  2. 라면을 넣는다
  3. 3분 기다린다
  4. 완성!

중첩된 목록
