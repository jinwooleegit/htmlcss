HTML 튜토리얼

HTML이란 무엇인가요?

HTML(HyperText Markup Language)은 웹 페이지의 구조를 만드는 마크업 언어입니다. HTML은 웹 브라우저에게 콘텐츠를 어떻게 표시할지 알려주는 역할을 합니다.

💡 알아두면 좋은 점

  • HTML은 프로그래밍 언어가 아닌 마크업 언어입니다
  • 모든 웹사이트의 기본 구조를 담당합니다
  • 배우기 쉽고 즉시 결과를 확인할 수 있습니다

간단한 HTML 예제

<!DOCTYPE html>
<html>
<head>
    <title>내 첫 번째 웹페이지</title>
</head>
<body>
    <h1>안녕하세요!</h1>
    <p>이것은 내 첫 번째 HTML 페이지입니다.</p>
</body>
</html>

HTML 문서의 기본 구조

모든 HTML 문서는 다음과 같은 기본 구조를 가집니다:

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <!-- 여기에 콘텐츠가 들어갑니다 -->
</body>
</html>
<!DOCTYPE html>

HTML5 문서임을 선언합니다

<html>

HTML 문서의 루트 요소입니다

<head>

문서의 메타정보를 포함합니다

<body>

실제 웹페이지에 표시될 콘텐츠를 포함합니다

자주 사용하는 HTML 태그들

<h1> ~ <h6>

제목 태그 (크기 순서대로)

<h1>가장 큰 제목</h1>

<p>

문단(paragraph) 태그

<p>이것은 문단입니다.</p>

<br>

줄바꿈 태그

첫 번째 줄<br>두 번째 줄

<div>

구역을 나누는 컨테이너 태그

<div>콘텐츠 그룹</div>

<span>

인라인 요소를 그룹화하는 태그

<span>특별한 텍스트</span>

<strong>

중요한 텍스트 (굵게 표시)

<strong>중요한 내용</strong>
