Cursor AI란?

AI 기반 코드 에디터

Cursor는 GPT-4를 기반으로 한 AI 코딩 도구로, 자연어로 요청하면 자동으로 코드를 생성하고 수정해주는 혁신적인 에디터입니다.

이 과정에서 배울 내용

  • Cursor AI 설치 및 기본 설정
  • AI를 활용한 HTML/CSS 코드 생성
  • 프롬프트 작성 기법과 팁
  • 반응형 웹사이트 제작 과정
  • AI와 함께하는 디버깅 및 최적화

실제 웹사이트 제작 과정

1단계: 기본 구조 생성

프롬프트:

"카페 홈페이지를 만들어줘. 헤더, 메인 콘텐츠, 푸터가 있는 기본 HTML 구조로 만들어줘."

생성된 HTML 구조

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>카페 홈페이지</title>
</head>
<body>
    <header>
        <nav>카페 로고 및 메뉴</nav>
    </header>
    <main>
        <section>메인 콘텐츠</section>
    </main>
    <footer>
        <p>카페 정보</p>
    </footer>
</body>
</html>

2단계: 스타일링 추가

프롬프트:

"카페 느낌의 따뜻한 색상으로 CSS를 만들어줘. 브라운 계열 색상을 사용하고 모던한 디자인으로 해줘."

생성된 CSS 스타일

body {
    font-family: 'Arial', sans-serif;
    margin: 0;
    background-color: #f5f3f0;
    color: #3c2415;
}

header {
    background-color: #8b4513;
    color: white;
    padding: 1rem;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

main {
    max-width: 1200px;
    margin: 2rem auto;
    padding: 0 1rem;
}

3단계: 반응형 디자인

프롬프트:

"모바일에서도 잘 보이도록 반응형 CSS를 추가해줘. 768px 이하에서는 메뉴가 햄버거 버튼으로 바뀌게 해줘."

반응형 CSS 추가

@media (max-width: 768px) {
    .nav-menu {
        display: none;
        flex-direction: column;
        position: absolute;
        top: 100%;
        left: 0;
        width: 100%;
        background-color: #8b4513;
    }
    
    .hamburger {
        display: block;
        cursor: pointer;
    }
    
    .nav-menu.active {
        display: flex;
    }
}

고급 기능과 최적화

JavaScript 인터랙션 추가

프롬프트:

"이미지 슬라이더를 만들어줘. 자동으로 3초마다 이미지가 바뀌고, 좌우 버튼으로도 조작할 수 있게 해줘."

생성된 JavaScript

class ImageSlider {
    constructor(container) {
        this.container = container;
        this.images = container.querySelectorAll('img');
        this.currentIndex = 0;
        this.init();
    }
    
    init() {
        this.createControls();
        this.autoPlay();
    }
    
    nextImage() {
        this.currentIndex = (this.currentIndex + 1) % this.images.length;
        this.updateSlider();
    }
    
    autoPlay() {
        setInterval(() => this.nextImage(), 3000);
    }
}

SEO 최적화

프롬프트:

"SEO에 좋도록 메타 태그들을 추가해줘. 카페 홈페이지에 맞는 키워드와 설명을 포함해줘."

SEO 메타 태그

<meta name="description" content="따뜻한 분위기의 카페, 신선한 원두와 수제 디저트를 제공합니다.">
<meta name="keywords" content="카페, 커피, 디저트, 원두, 핸드드립">
<meta property="og:title" content="따뜻한 카페 - 신선한 원두와 수제 디저트">
<meta property="og:description" content="편안한 분위기에서 즐기는 프리미엄 커피">
<meta property="og:image" content="cafe-main-image.jpg">

성능 최적화

프롬프트:

"웹사이트 로딩 속도를 빠르게 하려면 어떻게 해야 해? 이미지 최적화와 CSS 압축도 해줘."
