Cursor AI로 웹사이트 만들기

AI와 함께하는 효율적인 웹 개발 - 코딩 지식 없이도 전문적인 웹사이트 제작

Cursor AI 설치 → 핵심 기능 학습 → 실제 웹사이트 제작

Cursor AI란 무엇인가요?

Cursor AI = AI가 내장된 코드 에디터

자연어 코딩

평소 말하듯이 요청하면 AI가 코드를 자동으로 생성

예: "빨간색 버튼 만들어줘" → 완벽한 CSS 코드 생성
실시간 수정

코드를 선택하고 수정 요청하면 즉시 개선

예: 기존 코드를 선택하고 "더 예쁘게 만들어줘" 요청
전체 프로젝트 이해

여러 파일을 동시에 수정하고 일관된 스타일 적용

예: "전체 사이트를 다크모드로 변경해줘" → 모든 파일 자동 수정

실제 작업 비교

기존 방식 vs Cursor AI 방식

기존 방식 (5시간)
  1. HTML 문법 공부
  2. CSS 속성 찾아보기
  3. 코드 작성
  4. 오류 수정
  5. 디자인 조정
  6. 반응형 적용

시간이 오래 걸리고 복잡함

Cursor AI 방식 (30분)
  1. "회사 소개 페이지 만들어줘"
  2. "헤더에 로고 추가해줘"
  3. "색상을 파란색으로 바꿔줘"
  4. "모바일에서도 잘 보이게 해줘"

자연어로 간단하게 완성

핵심 인사이트: 복잡한 코딩 지식 없이도 AI와 대화하듯 웹사이트를 만들 수 있습니다!

쉬운 비유로 이해하기

Cursor AI = 전문 비서

  • 일반 에디터 = 혼자서 모든 걸 직접 해야 하는 상황
  • Cursor AI = 전문 비서가 옆에서 즉시 도와주는 상황
핵심 팁: 명확하고 구체적으로 요청할수록 더 정확한 결과를 얻을 수 있어요!

왜 Cursor AI를 사용해야 할까요?

학습 시간 단축

문법을 외우는 대신 바로 실무에 집중할 수 있음

실수 방지

AI가 문법 오류와 최적화를 자동으로 처리

창의성 향상

기술적 제약에서 벗어나 아이디어 구현에 집중

지속적 학습

AI가 생성한 코드를 보며 자연스럽게 실력 향상

Cursor AI 설치하기

단계별 설치 가이드

1

공식 웹사이트 방문

cursor.sh에 접속하여 다운로드 버튼 클릭

팁: Windows, Mac, Linux 모두 지원합니다!

2

설치 파일 실행

다운로드된 파일을 실행하고 안내에 따라 설치

주의: 관리자 권한이 필요할 수 있습니다

3

계정 생성 및 로그인

이메일로 계정을 만들고 Cursor AI에 로그인

혜택: 무료 플랜으로도 충분히 사용 가능합니다

4

첫 번째 프로젝트 생성

새 폴더를 만들고 Cursor로 열어보기

1. 바탕화면에 "my-website" 폴더 생성
2. Cursor AI 실행
3. File → Open Folder → my-website 선택
4. 준비 완료!

핵심 기능 3가지 마스터하기

Cursor AI의 3가지 핵심 기능

1

Chat 기능 (Ctrl+L)

대화형

사용법: Ctrl+L을 누르고 자연어로 요청

프롬프트

"빨간색 버튼을 만들어줘. 클릭하면 알림창이 뜨게 해줘"

결과

<button onclick="alert('버튼이 클릭되었습니다!')" 
        style="background: #e74c3c; color: white; 
               padding: 12px 24px; border: none; 
               border-radius: 5px; cursor: pointer;">
    클릭하세요
</button>
장점: 새로운 코드 생성에 최적
사용 시기: 처음부터 만들 때
2

Quick Edit (Ctrl+K)

즉시 수정

사용법: 코드 선택 → Ctrl+K → 수정 요청

기존 코드

<h1>안녕하세요</h1>

요청: "제목을 더 예쁘게 만들어줘"

↓

개선된 코드

<h1 style="color: #2c3e50; font-size: 2.5em; 
           text-align: center; margin: 20px 0; 
           font-weight: 300;">
    안녕하세요
</h1>
장점: 기존 코드 개선에 최적
사용 시기: 이미 있는 코드를 수정할 때
3

Composer (Ctrl+I)

프로젝트 전체

사용법: Ctrl+I를 누르고 프로젝트 전체 작업 요청

프롬프트

"전체 웹사이트를 다크모드로 바꿔줘. 배경은 어둡게, 텍스트는 밝게 해줘"

결과

✅ index.html 수정됨
✅ style.css 수정됨  
✅ about.html 수정됨
✅ contact.html 수정됨

모든 파일이 일관된 다크모드로 변경!
장점: 여러 파일 동시 수정
사용 시기: 프로젝트 전체 변경할 때

기능별 사용 가이드

기능 단축키 사용 시기 장점
Chat Ctrl+L 새로운 코드 생성 자연어 대화
Quick Edit Ctrl+K 기존 코드 수정 즉시 개선
Composer Ctrl+I 프로젝트 전체 작업 일관된 변경

실제 웹사이트 만들기 실습

프로젝트: 개인 포트폴리오 사이트 제작

Cursor AI와 함께 단계별로 완전한 웹사이트를 만들어보겠습니다.

1

기본 HTML 구조 생성

Chat (Ctrl+L)에서 요청:

"개인 포트폴리오 웹사이트의 기본 HTML 구조를 만들어줘. 
헤더, 소개, 프로젝트, 연락처 섹션이 필요해"
2

스타일링 추가

추가 요청:

"CSS 파일을 만들어서 깔끔하고 현대적인 디자인을 적용해줘. 
색상은 파란색 계열로, 폰트는 깔끔하게 해줘"
3

반응형 디자인 적용

Composer (Ctrl+I)에서 요청:

"모바일에서도 잘 보이도록 반응형 디자인을 적용해줘. 
태블릿과 스마트폰에서 레이아웃이 자동으로 조정되게 해줘"
4

인터랙션 추가

추가 기능 요청:

"버튼에 호버 효과를 추가하고, 
스크롤할 때 부드럽게 섹션이 나타나는 애니메이션을 만들어줘"

완성된 결과물

완전한 반응형 포트폴리오 웹사이트

현대적이고 깔끔한 디자인

모바일 최적화

부드러운 애니메이션 효과

축하합니다! AI와 함께 전문적인 웹사이트를 완성했습니다!

효과적인 프롬프트 작성법

AI와 소통하는 법

구체적으로 요청하기

나쁜 예시

"버튼 만들어줘"

좋은 예시

"파란색 배경에 흰색 텍스트인 버튼을 만들어줘. 크기는 중간 정도로, 모서리는 둥글게 해줘"

목적과 맥락 설명하기

나쁜 예시

"헤더 만들어줘"

좋은 예시

"카페 웹사이트의 헤더를 만들어줘. 로고, 메뉴(메뉴, 위치, 연락처), 주문하기 버튼이 필요해"

단계별로 요청하기

효과적인 순서
  1. "기본 HTML 구조 만들어줘"
  2. "CSS로 스타일링 해줘"
  3. "반응형으로 만들어줘"
  4. "애니메이션 추가해줘"

주의사항 및 실무 팁

주의사항

코드 검토하기:

AI가 생성한 코드도 항상 검토하고 이해하려고 노력하세요

백업 유지하기:

중요한 변경 전에는 항상 파일을 백업해두세요

단계별 진행:

한 번에 너무 많은 것을 요청하지 말고 단계별로 진행하세요

실무 팁

생산성 향상

  • 자주 사용하는 요청은 템플릿으로 저장
  • 단축키를 외워서 빠르게 접근
  • 프로젝트 구조를 미리 계획

디자인 개선

  • 색상 코드를 구체적으로 지정
  • 레퍼런스 사이트를 언급
  • 사용자 경험을 고려한 요청

문제 해결

  • 오류 메시지를 AI에게 보여주기
  • 원하는 결과와 현재 상황 설명
  • 다양한 방법으로 재요청
