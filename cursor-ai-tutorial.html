Cursor AI 설치하기

간단한 4단계로 시작하는 AI 코딩

1

공식 웹사이트 방문

cursor.sh 웹사이트에 접속하여 시스템에 맞는 버전을 확인합니다.

지원 환경:

Windows, Mac, Linux 모두 지원됩니다.

2

설치 파일 다운로드

다운로드 버튼을 클릭하여 최신 버전의 설치 파일을 받습니다.

참고:

파일 크기는 약 100MB 내외입니다.

3

설치 진행

다운로드된 설치 파일을 실행하고 안내에 따라 설치를 진행합니다.

주의:

설치 과정에서 관리자 권한이 필요할 수 있습니다.

4

초기 설정

Cursor AI를 처음 실행하고 기본 설정을 완료합니다.

팁:

GitHub 계정으로 로그인하면 더 많은 기능을 사용할 수 있습니다.

주의사항 및 실무 팁

주의사항

1코드 검토하기

AI가 생성한 코드도 항상 검토하고 이해하려고 노력하세요

2백업 유지하기

중요한 변경 전에는 항상 파일을 백업해두세요

3단계별 진행

한 번에 너무 많은 것을 요청하지 말고 단계별로 진행하세요

실무 팁

1생산성 향상

  • 자주 사용하는 요청은 템플릿으로 저장
  • 단축키를 외워서 빠르게 접근
  • 프로젝트 구조를 미리 계획

2디자인 개선

  • 색상 코드를 구체적으로 지정
  • 레퍼런스 사이트를 언급
  • 사용자 경험을 고려한 요청

3문제 해결

  • 오류 메시지를 AI에게 보여주기
  • 원하는 결과와 현재 상황 설명
  • 다양한 방법으로 재요청
