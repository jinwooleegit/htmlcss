CSS 튜토리얼

CSS란 무엇인가요?

CSS(Cascading Style Sheets)는 HTML 문서의 스타일을 정의하는 언어입니다. 웹 페이지의 색상, 레이아웃, 폰트 등 시각적 표현을 담당합니다.

💡 CSS의 장점

  • HTML과 디자인을 분리하여 관리가 용이합니다
  • 하나의 CSS 파일로 여러 HTML 페이지를 스타일링할 수 있습니다
  • 반응형 디자인을 구현할 수 있습니다

간단한 CSS 예제

HTML
<h1 class="title">안녕하세요!</h1>
<p class="description">이것은 스타일이 적용된 텍스트입니다.</p>
CSS
.title {
    color: blue;
    font-size: 2em;
}

.description {
    color: gray;
    font-style: italic;
}

CSS 선택자

선택자는 스타일을 적용할 HTML 요소를 선택하는 방법입니다.

요소 선택자

HTML 태그명으로 선택

h1 { color: red; }

클래스 선택자

class 속성으로 선택

.my-class { color: blue; }

ID 선택자

id 속성으로 선택

#my-id { color: green; }

전체 선택자

모든 요소를 선택

* { margin: 0; }

CSS 박스 모델

모든 HTML 요소는 박스 형태로 구성되며, 다음과 같은 구조를 가집니다:

Margin
Border
Padding
Content
Content

실제 내용이 들어가는 영역

Padding

내용과 테두리 사이의 여백

Border

요소의 테두리

Margin

다른 요소와의 외부 여백

박스 모델 CSS 예제

.box {
    width: 200px;
    height: 100px;
    padding: 20px;
    border: 2px solid black;
    margin: 10px;
}
