제목 태그 사용 가이드라인

  • h1: 페이지의 메인 주제 (SEO에 중요)
  • h2: 주요 섹션 구분
  • h3-h6: 하위 구조 표현
  • 제목 레벨을 건너뛰지 마세요 (h1 → h3 ❌)

텍스트 서식 태그 심화

    <strong>중요한 내용</strong> (굵게, 의미적 강조)<br>

    <em>강조하고 싶은 내용</em> (기울임, 의미적 강조)<br>

    <small>작은 글씨 (저작권 등)</small><br>
    <code>console.log('코드')</code>
</p>

이미지 완전 활용법

<!-- 기본 이미지 -->
<img src="photo.jpg" alt="사진 설명" width="300" height="200">

<!-- 반응형 이미지 -->
<img src="photo-large.jpg" 
     srcset="photo-small.jpg 480w, photo-medium.jpg 800w, photo-large.jpg 1200w"
     sizes="(max-width: 480px) 100vw, (max-width: 800px) 50vw, 25vw"
     alt="반응형 이미지">

<!-- 이미지 맵 (클릭 가능한 영역) -->
<img src="worldmap.jpg" alt="세계지도" usemap="#worldmap">
<map name="worldmap">
    <area shape="rect" coords="0,0,100,100" href="korea.html" alt="한국">
    <area shape="circle" coords="200,200,50" href="usa.html" alt="미국">
</map>

<!-- 이미지와 캡션 -->
<figure>
    <img src="sunset.jpg" alt="아름다운 석양">
    <figcaption>제주도에서 본 아름다운 석양</figcaption>
</figure>

🎯 이미지 최적화 팁

  • alt 속성: 항상 의미 있는 설명 작성
  • 파일 크기: 웹용으로 압축 (보통 100KB 이하)
  • 파일 형식: JPEG(사진), PNG(투명배경), WebP(최신)
  • 로딩 최적화: loading="lazy" 속성 활용

📊 테이블 구조 이해하기

  • caption: 테이블 제목
  • thead: 헤더 영역
  • tbody: 데이터 영역
  • tfoot: 푸터 영역 (합계 등)
  • colspan: 열 병합
  • rowspan: 행 병합

폼(Forms) 완전 정복

회원가입 폼 만들기

<form action="/signup" method="post">
    <fieldset>
        <legend>개인정보</legend>
        
        <label for="username">사용자명 (필수):</label>
        <input type="text" id="username" name="username" required>
        
        <label for="email">이메일 (필수):</label>
        <input type="email" id="email" name="email" required>
        
        <label for="password">비밀번호 (필수):</label>
        <input type="password" id="password" name="password" 
               minlength="8" required>
        
        <label for="birthdate">생년월일:</label>
        <input type="date" id="birthdate" name="birthdate">
    </fieldset>
    
    <fieldset>
        <legend>선호사항</legend>
        
        <label>성별:</label>
        <input type="radio" id="male" name="gender" value="male">
        <label for="male">남성</label>
        <input type="radio" id="female" name="gender" value="female">
        <label for="female">여성</label>
        
        <label>관심분야 (복수선택 가능):</label>
        <input type="checkbox" id="html" name="interests" value="html">
        <label for="html">HTML</label>
        <input type="checkbox" id="css" name="interests" value="css">
        <label for="css">CSS</label>
        <input type="checkbox" id="js" name="interests" value="js">
        <label for="js">JavaScript</label>
        
        <label for="country">국가:</label>
        <select id="country" name="country">
            <option value="">선택하세요</option>
            <option value="kr">한국</option>
            <option value="us">미국</option>
            <option value="jp">일본</option>
        </select>
        
        <label for="bio">자기소개:</label>
        <textarea id="bio" name="bio" rows="4" cols="50" 
                  placeholder="간단한 자기소개를 입력하세요"></textarea>
    </fieldset>
    
    <fieldset>
        <legend>약관 동의</legend>
        <input type="checkbox" id="terms" name="terms" required>
        <label for="terms">이용약관에 동의합니다 (필수)</label>
        
        <input type="checkbox" id="newsletter" name="newsletter">
        <label for="newsletter">뉴스레터 수신에 동의합니다 (선택)</label>
    </fieldset>
    
    <button type="submit">회원가입</button>
    <button type="reset">초기화</button>
</form>

선호사항

폼 구성 요소 설명

  • fieldset + legend: 관련 입력 요소들을 그룹화
  • label: 입력 요소의 설명 (for 속성으로 연결)
  • input type="text": 일반 텍스트 입력
  • input type="email": 이메일 형식 검증
  • input type="password": 비밀번호 (숨김 표시)
  • input type="radio": 단일 선택
  • input type="checkbox": 다중 선택
  • select + option: 드롭다운 목록
  • textarea: 여러 줄 텍스트 입력

멀티미디어 활용

오디오와 비디오

<!-- 오디오 -->
<audio controls>
    <source src="music.mp3" type="audio/mpeg">
    <source src="music.ogg" type="audio/ogg">
    브라우저가 오디오를 지원하지 않습니다.
</audio>

<!-- 비디오 -->
<video width="400" height="300" controls poster="thumbnail.jpg">
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.webm" type="video/webm">
    <track src="subtitles.vtt" kind="subtitles" srclang="ko" label="한국어">
    브라우저가 비디오를 지원하지 않습니다.
</video>

<!-- YouTube 임베드 -->
<iframe width="560" height="315" 
        src="https://www.youtube.com/embed/VIDEO_ID" 
        title="YouTube video player" 
        frameborder="0" 
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" 
        allowfullscreen>
</iframe>

멀티미디어 최적화 팁

  • 여러 포맷 제공: 브라우저 호환성을 위해
  • controls 속성: 재생 컨트롤 표시
  • poster 속성: 비디오 썸네일 이미지
  • 자막 지원: track 태그로 접근성 향상
  • 적절한 크기: 파일 크기와 품질의 균형
