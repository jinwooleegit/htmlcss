🌱 HTML 초급편

HTML이 무엇인지부터 기본 문법까지 차근차근 배워보세요

초급편 → 중급편 → 고급편

HTML이란 무엇인가요?

HTML = HyperText Markup Language

HyperText

링크로 연결된 텍스트

예: 네이버에서 "영화"를 클릭하면 영화 페이지로 이동하는 것처럼, 텍스트를 클릭해서 다른 페이지로 갈 수 있는 기능입니다.
Markup

구조를 표시하는 방법

예: 워드에서 제목을 크게 만들고, 본문을 작게 만드는 것처럼, 웹페이지의 구조를 표시하는 방법입니다.
Language

컴퓨터가 이해하는 언어

예: 우리가 한국어로 말하듯이, 브라우저가 이해할 수 있는 특별한 언어입니다.

실제 예시로 이해하기

HTML 없이는 웹페이지가 어떻게 보일까요?

HTML 없는 텍스트:
안녕하세요 이것은 제목입니다 이것은 본문입니다 이것은 링크입니다
HTML로 구조화된 텍스트:

안녕하세요

이것은 제목입니다

이것은 본문입니다

이것은 링크입니다

인사이트: HTML이 있으면 같은 내용이지만 훨씬 읽기 쉽고 구조적으로 보입니다!

쉬운 비유로 이해하기

웹페이지를 집이라고 생각해보세요:

  • HTML = 집의 뼈대 (기둥, 벽, 방)
  • CSS = 집의 인테리어 (색깔, 가구, 장식)
  • JavaScript = 집의 전자제품 (조명, 에어컨, TV)

왜 HTML을 배워야 할까요?

모든 웹사이트의 기초

구글, 페이스북, 유튜브 모두 HTML로 만들어졌어요

배우기 쉬움

프로그래밍 경험이 없어도 하루만에 기초를 익힐 수 있어요

취업에 도움

웹 개발자, 디자이너, 마케터 모두에게 필수 기술이에요

즉시 결과 확인

코드를 작성하면 바로 웹페이지에서 결과를 볼 수 있어요

HTML 문법의 기초

HTML의 기본 구조: 태그(Tag)

< 태그이름 > 내용 </ 태그이름 >
여는 태그: <태그이름>
내용: 실제 표시될 텍스트
닫는 태그: </태그이름>

HTML 문법 규칙

대소문자 구분 안함

<H1>과 <h1>은 같음

태그는 중첩 가능

<div><p>내용</p></div>

빈 태그도 있음

<br>, <img> 등

들여쓰기 권장

코드를 읽기 쉽게 만듦

초보자가 자주 하는 실수들

틀린 예시:
<h1>제목</h2>

여는 태그와 닫는 태그가 다름

올바른 예시:
<h1>제목</h1>

여는 태그와 닫는 태그가 같음

틀린 예시:
<p>문단<p>

닫는 태그에 슬래시(/) 없음

올바른 예시:
<p>문단</p>

닫는 태그에 슬래시(/) 있음

틀린 예시:
<div><p>내용</div></p>

태그 순서가 잘못됨

올바른 예시:
<div><p>내용</p></div>

나중에 연 태그를 먼저 닫음

첫 번째 실습

아래 코드를 따라 써보세요:

<h1>안녕하세요!</h1>
<p>이것은 내 첫 번째 HTML 태그입니다.</p>
결과:

안녕하세요!

이것은 내 첫 번째 HTML 태그입니다.

실습 팁:
  • 메모장이나 텍스트 에디터에 위 코드를 입력하세요
  • 파일을 test.html로 저장하세요
  • 저장한 파일을 더블클릭하면 브라우저에서 결과를 볼 수 있어요
  • 코드를 수정하고 새로고침하면 변경사항이 반영됩니다

단계별 따라하기

1
메모장 열기

윈도우: 시작 → 메모장 검색
맥: 응용프로그램 → 텍스트편집기

2
코드 입력

위의 HTML 코드를 정확히 입력하세요. 대소문자와 괄호를 정확히!

3
파일 저장

Ctrl+S (윈도우) 또는 Cmd+S (맥)를 누르고 'my-first-page.html'로 저장

4
브라우저에서 열기

저장한 파일을 더블클릭하거나 브라우저로 드래그해서 결과 확인!

HTML 문서의 기본 구조

모든 HTML 문서는 이런 구조를 가져요

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <!-- 여기에 콘텐츠가 들어갑니다 -->
</body>
</html>

<!DOCTYPE html>

역할: "이 문서는 HTML5로 작성되었습니다"라고 브라우저에게 알려줍니다

위치: 문서의 맨 첫 줄

<html>

역할: HTML 문서의 시작과 끝을 나타냅니다

속성: lang="ko"는 한국어 문서임을 표시합니다

<head>

역할: 웹페이지의 정보를 담습니다 (사용자에게는 보이지 않음)

포함 내용: 제목, 인코딩, CSS 링크 등

<body>

역할: 실제 웹페이지에 표시될 모든 내용을 담습니다

포함 내용: 텍스트, 이미지, 링크 등 모든 콘텐츠

<head> 태그 안에 들어가는 중요한 것들

<meta charset="UTF-8">

한글이 깨지지 않게 하는 인코딩 설정

왜 필요한가요? 컴퓨터는 문자를 숫자로 저장하는데, UTF-8은 전 세계 모든 언어를 표현할 수 있는 방식입니다. 이게 없으면 한글이 ????????나 □□□□로 보일 수 있어요.
<title>페이지 제목</title>

브라우저 탭에 표시되는 제목

어디에 보이나요? 브라우저 탭, 북마크 이름, 검색결과에서 보이는 제목입니다. 구글에서 검색했을 때 파란색으로 보이는 그 제목이에요!
<meta name="viewport" content="width=device-width">

모바일에서 제대로 보이게 하는 설정

왜 필요한가요? 이게 없으면 모바일에서 웹페이지가 너무 작게 보여서 확대해야 합니다. 이 설정으로 모바일 화면에 맞게 자동으로 크기가 조절돼요.

완전한 HTML 페이지 만들어보기

이제 모든 내용을 합쳐서 완전한 웹페이지를 만들어볼까요?

완전한 HTML 문서:

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>내 첫 번째 웹페이지</title>
</head>
<body>
    <h1>안녕하세요! 저는 홍길동입니다.</h1>
    <p>이것은 제가 만든 첫 번째 웹페이지입니다.</p>
    <p>HTML을 배우는 것이 정말 재미있어요!</p>
</body>
</html>

이 코드를 실행하면:

브라우저 탭: "내 첫 번째 웹페이지"라고 표시됩니다
페이지 내용:

안녕하세요! 저는 홍길동입니다.

이것은 제가 만든 첫 번째 웹페이지입니다.

HTML을 배우는 것이 정말 재미있어요!

숙제: 나만의 소개 페이지 만들기

다음 내용을 포함해서 여러분만의 소개 페이지를 만들어보세요:

  • 여러분의 이름을 h1 태그로 만들기
  • 자기소개를 p 태그로 3개 이상 작성하기
  • 페이지 제목을 여러분 이름으로 바꾸기
  • 파일명을 "내이름.html"로 저장하기
예시: "김철수.html" 파일에 김철수에 대한 소개 내용 작성
