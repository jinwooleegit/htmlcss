🤔 HTML이란 무엇인가요?

HTML = HyperText Markup Language

HyperText

링크로 연결된 텍스트

Markup

구조를 표시하는 방법

Language

컴퓨터가 이해하는 언어

🏠 쉬운 비유로 이해하기

웹페이지를 집이라고 생각해보세요:

  • HTML = 집의 뼈대 (기둥, 벽, 방)
  • CSS = 집의 인테리어 (색깔, 가구, 장식)
  • JavaScript = 집의 전자제품 (조명, 에어컨, TV)

🎯 왜 HTML을 배워야 할까요?

🌐

모든 웹사이트의 기초

구글, 페이스북, 유튜브 모두 HTML로 만들어졌어요

⚡

배우기 쉬움

프로그래밍 경험이 없어도 하루만에 기초를 익힐 수 있어요

💼

취업에 도움

웹 개발자, 디자이너, 마케터 모두에게 필수 기술이에요

🚀

즉시 결과 확인

코드를 작성하면 바로 웹페이지에서 결과를 볼 수 있어요

📝 HTML 문법의 기초

HTML의 기본 구조: 태그(Tag)

< 태그이름 > 내용 </ 태그이름 >
여는 태그: <태그이름>
내용: 실제 표시될 텍스트
닫는 태그: </태그이름>

🔤 HTML 문법 규칙

✅
대소문자 구분 안함

<H1>과 <h1>은 같음

✅
태그는 중첩 가능

<div><p>내용</p></div>

✅
빈 태그도 있음

<br>, <img> 등

✅
들여쓰기 권장

코드를 읽기 쉽게 만듦

✏️ 첫 번째 실습

아래 코드를 따라 써보세요:

<h1>안녕하세요!</h1>
<p>이것은 내 첫 번째 HTML 태그입니다.</p>
결과:

안녕하세요!

이것은 내 첫 번째 HTML 태그입니다.

🏗️ HTML 문서의 기본 구조

모든 HTML 문서는 이런 구조를 가져요

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <!-- 여기에 콘텐츠가 들어갑니다 -->
</body>
</html>

<!DOCTYPE html>

역할: "이 문서는 HTML5로 작성되었습니다"라고 브라우저에게 알려줍니다

위치: 문서의 맨 첫 줄

<html>

역할: HTML 문서의 시작과 끝을 나타냅니다

속성: lang="ko"는 한국어 문서임을 표시합니다

<head>

역할: 웹페이지의 정보를 담습니다 (사용자에게는 보이지 않음)

포함 내용: 제목, 인코딩, CSS 링크 등

<body>

역할: 실제 웹페이지에 표시될 모든 내용을 담습니다

포함 내용: 텍스트, 이미지, 링크 등 모든 콘텐츠

<head> 태그 안에 들어가는 중요한 것들

<meta charset="UTF-8">

한글이 깨지지 않게 하는 인코딩 설정

<title>페이지 제목</title>

브라우저 탭에 표시되는 제목

<meta name="viewport" content="width=device-width">

모바일에서 제대로 보이게 하는 설정
