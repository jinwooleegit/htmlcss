1. 웹 호스팅 기초 이해 🏠

웹 호스팅이란?

웹 호스팅은 여러분의 웹사이트 파일들을 인터넷에 연결된 서버에 저장하여, 전 세계 사람들이 접근할 수 있게 하는 서비스입니다.

🖥️ 서버란?

24시간 켜져있는 컴퓨터로, 여러분의 웹사이트 파일을 저장하고 방문자들에게 제공합니다.

🌐 도메인이란?

웹사이트의 주소입니다. 예: www.mywebsite.com

📡 DNS란?

도메인 이름을 서버의 실제 주소(IP)로 변환해주는 시스템입니다.

2. 호스팅 서비스 종류 📊

호스팅 종류 가격 난이도 적합한 용도 추천 서비스
무료 호스팅 무료 쉬움 연습, 포트폴리오 GitHub Pages, Netlify
공유 호스팅 월 3,000-10,000원 쉬움 개인 웹사이트, 블로그 카페24, 가비아, 호스팅케이알
VPS 호스팅 월 10,000-50,000원 중간 중소규모 비즈니스 AWS EC2, DigitalOcean
전용 서버 월 100,000원+ 어려움 대규모 웹사이트 전문 호스팅 업체

🎯 초보자 추천

1단계: GitHub Pages (무료) → 2단계: 카페24/가비아 공유호스팅 → 3단계: AWS/DigitalOcean VPS

3. 도메인 구입 가이드 🛒

1

도메인 이름 결정

  • 기억하기 쉬운 이름
  • 브랜드나 목적과 관련된 이름
  • 너무 길지 않게 (15자 이내 권장)
  • 특수문자 사용 금지
2

도메인 확장자 선택

.com (연 15,000원) .co.kr (연 20,000원) .net (연 18,000원) .org (연 16,000원)
3

도메인 등록업체 선택

🇰🇷 국내 업체
  • 가비아 (gabia.com)
  • 후이즈 (whois.co.kr)
  • 카페24 (cafe24.com)
🌍 해외 업체
  • GoDaddy (godaddy.com)
  • Namecheap (namecheap.com)
  • Google Domains

💡 도메인 구입 체크리스트

✅ 원하는 도메인 이름 사용 가능한지 확인
✅ 가격 비교 (등록비 + 갱신비)
✅ DNS 관리 기능 포함 여부
✅ 개인정보 보호 서비스 포함 여부
✅ 고객 지원 서비스 품질
✅ 자동 갱신 설정

4. 실제 배포 단계 🚀

🏢 카페24 공유호스팅 배포

1단계: 호스팅 신청
  • cafe24.com 접속
  • 웹호스팅 → 리눅스 호스팅 선택
  • 요금제 선택 (월 990원부터)
  • 결제 및 계정 생성
2단계: 도메인 연결
  • 카페24 관리자 페이지 로그인
  • 도메인 관리 → 도메인 추가
  • 구입한 도메인 입력
  • DNS 설정 완료
3단계: 파일 업로드
FTP 정보 확인:
- FTP 주소: 호스팅 업체에서 제공
- 사용자명: 계정명
- 비밀번호: 설정한 비밀번호
- 포트: 21 (일반) 또는 22 (SFTP)
  • FileZilla 다운로드 및 설치
  • FTP 접속 정보 입력
  • HTML, CSS, JS 파일 업로드
  • public_html 폴더에 업로드

☁️ AWS EC2 배포

1단계: EC2 인스턴스 생성
  • AWS 계정 생성 (aws.amazon.com)
  • EC2 서비스 선택
  • Ubuntu 서버 선택
  • t2.micro (프리티어) 선택
2단계: 웹서버 설치
# Apache 설치
sudo apt update
sudo apt install apache2

# 방화벽 설정
sudo ufw allow 'Apache Full'

# 서비스 시작
sudo systemctl start apache2
sudo systemctl enable apache2
3단계: 파일 업로드
# SCP로 파일 업로드
scp -i key.pem -r website/* ubuntu@your-ip:/var/www/html/

# 또는 서버에서 직접
sudo cp -r /home/ubuntu/website/* /var/www/html/
sudo chown -R www-data:www-data /var/www/html/

⚡ Netlify 배포 (추천)

1단계: Netlify 계정 생성
  • netlify.com 접속
  • GitHub 계정으로 로그인
  • 무료 계정 생성
2단계: 사이트 배포
  • "New site from Git" 클릭
  • GitHub 저장소 선택
  • 배포 설정 확인
  • 자동 배포 완료!
3단계: 커스텀 도메인 연결
  • Site settings → Domain management
  • "Add custom domain" 클릭
  • 구입한 도메인 입력
  • DNS 설정 완료

7. 비용 계획 및 관리 💰
