01

디자이너가 알아야 할 JavaScript 기초

JavaScript의 역할

동적 요소 제어

사용자 동작에 반응하여 웹페이지의 내용과 스타일을 동적으로 변경

인터랙션 처리

클릭, 스크롤, 드래그 등 다양한 사용자 상호작용 구현

애니메이션 구현

부드럽고 자연스러운 움직임과 전환 효과 생성

데이터 처리

서버로부터 데이터를 받아와 동적으로 화면에 표시

주요 개념과 용어

1. 이벤트 (Events)

사용자 동작 
// 클릭, 스크롤, 마우스오버 등 사용자 동작
element.addEventListener('click', function() {
    // 동작 실행
});

사용자의 동작을 감지하고 반응하는 방법

2. DOM 조작

요소 제어 
// 요소 선택과 수정
document.querySelector('.my-element')
element.classList.add('active');

웹페이지의 요소를 선택하고 변경하는 방법

3. 애니메이션

움직임 
// CSS 트랜지션과 JavaScript 연동
element.style.transform = 'translateX(100px)';

요소에 움직임과 변화를 주는 방법

02

인터랙션 디자인 구현하기

자주 사용되는 인터랙션 패턴

스크롤 기반 애니메이션

window.addEventListener('scroll', () => {
    // 스크롤 위치에 따른 요소 애니메이션
});
  • 스크롤 위치 감지
  • 요소 등장/사라짐 효과
  • 패럴랙스 효과

모달 / 팝업

// 모달 열기/닫기 로직
const modal = document.querySelector('.modal');
modal.classList.toggle('active');
  • 부드러운 등장/사라짐
  • 배경 딤처리
  • 키보드 접근성

폼 인터랙션

// 폼 유효성 검사 및 피드백
form.addEventListener('submit', (e) => {
    e.preventDefault();
    // 폼 처리 로직
});
  • 실시간 유효성 검사
  • 시각적 피드백
  • 에러 메시지 표시

4. AI 도구를 활용한 작업 효율화

Cursor AI 활용하기

1. 인터랙션 코드 생성

// Cursor AI에게 요청하기
"버튼 호버시 부드럽게 확대되는 효과를 만들어줘"

AI가 생성한 코드를 바로 적용하고 수정할 수 있습니다.

2. 애니메이션 최적화

// 성능 개선 요청하기
"이 애니메이션 코드를 더 효율적으로 개선해줘"

AI가 성능과 호환성을 고려한 최적화된 코드를 제안합니다.

3. 문제 해결

// 디버깅 지원
"이 인터랙션이 Safari에서 작동하지 않아"

브라우저 호환성 문제나 버그를 AI가 분석하고 해결책을 제시합니다.

AI 기반 워크플로우

  1. 디자인 의도 설명 → AI가 초기 코드 생성
  2. 코드 리뷰 및 수정 요청 → AI가 최적화
  3. 문제점 발견 → AI가 디버깅 지원
  4. 최종 코드 완성 및 개발자 리뷰
