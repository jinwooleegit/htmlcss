JavaScript란 무엇인가요?

JavaScript = 웹을 살아있게 만드는 프로그래밍 언어

동적 프로그래밍 언어

웹페이지를 실시간으로 바꾸고, 사용자와 상호작용할 수 있게 함

예: 버튼을 클릭하면 내용이 바뀌거나, 입력창을 채우면 결과가 실시간으로 변하는 기능
클라이언트 사이드 언어

웹 브라우저에서 직접 실행되는 언어

예: 별도의 프로그램 설치 없이 웹 브라우저만 있으면 바로 실행 가능
웹 표준 언어

모든 브라우저에서 지원하는 공식 언어

HTML, CSS와 함께 웹의 3대 핵심 기술

실제 예시로 이해하기

JavaScript 없는 웹페이지 vs JavaScript 있는 웹페이지

JavaScript 없는 페이지

수량: 1 (변경 불가)

총 가격: 10,000원 (고정)
시간: 정적 표시
JavaScript 있는 페이지

수량: 1

총 가격: 10,000원
현재 시간:

인사이트: JavaScript가 있으면 웹페이지가 살아 움직이고, 사용자의 행동에 즉시 반응합니다!

쉬운 비유로 이해하기

웹페이지를 사람이라고 생각해보세요:

  • HTML = 뼈대와 장기 (구조)
  • CSS = 외모와 스타일 (피부, 옷)
  • JavaScript = 뇌와 근육 (생각하고 움직임)

JavaScript 없으면 아무리 예쁜 웹페이지도 '마네킹'에 불과해요!

왜 JavaScript를 배워야 할까요?

모든 웹사이트의 핵심

구글, 네이버, 유튜브, 페이스북 등 모든 사이트가 JavaScript로 동작

즉시 결과 확인

코드를 작성하면 바로 브라우저에서 결과를 볼 수 있음

취업과 실무 필수

웹 개발자, 프론트엔드, 백엔드, 앱 개발까지 필수 기술

배우기 상대적으로 쉬움

프로그래밍 입문자도 쉽게 시작할 수 있는 언어

JavaScript 코드는 어떻게 생겼을까요?

// 이것은 주석입니다
console.log("안녕하세요!");
let message = "JavaScript 배우기";
alert(message);

세미콜론: 문장의 끝을 표시 (선택사항)

JavaScript를 HTML에 추가하는 방법

3가지 방법으로 JavaScript를 추가할 수 있습니다

1

인라인 JavaScript

HTML 태그 내부 
<button onclick="alert('버튼 클릭!')">클릭하세요</button>
장점: 간단하고 빠름
단점: 코드가 길어지면 관리가 어려움
2

내부 JavaScript

HTML 파일 내부 
<script>
    console.log("내부 JavaScript 실행!");
    alert("Hello World!");
</script>
장점: HTML과 함께 관리 가능
단점: 재사용이 어려움
3

외부 JavaScript

추천 
<script src="script.js"></script>
장점: 재사용 가능, 관리 쉬움, 성능 좋음
단점: 파일이 하나 더 필요

실습 3: HTML에 JavaScript 추가하기

1

HTML 파일 만들기

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>JavaScript 실습</title>
</head>
<body>
    <h1>JavaScript 실습 페이지</h1>
    <button onclick="sayHello()">인사하기</button>
    
    <script>
        function sayHello() {
            alert("안녕하세요!");
        }
    </script>
</body>
</html>

위 코드를 메모장에 복사하고 test.html로 저장하세요!

2

브라우저에서 열기

저장한 HTML 파일을 더블클릭하여 브라우저에서 열어보세요!

3

버튼 클릭해보기

"인사하기" 버튼을 클릭하면 팝업이 나타납니다!

자주하는 실수와 해결법

❌ 실수 1

대소문자 구분 안하기

❌ 잘못된 코드
console.Log("Hello"); // Log는 대문자 L
✅ 올바른 코드
console.log("Hello"); // log는 소문자 l

해결법: JavaScript는 대소문자를 구분하므로 정확히 입력해야 합니다.

❌ 실수 2

따옴표 닫지 않기

❌ 잘못된 코드
alert("안녕하세요); // 따옴표 짝이 안 맞음
✅ 올바른 코드
alert("안녕하세요"); // 따옴표 짝이 맞음

해결법: 따옴표를 열었으면 반드시 같은 따옴표로 닫아야 합니다.

❌ 실수 3

함수 이름 오타내기

❌ 잘못된 코드
alret("Hello"); // alert 오타
✅ 올바른 코드
alert("Hello"); // alert 정확

해결법: 함수 이름을 정확히 외우고, 자동완성 기능을 활용하세요.
