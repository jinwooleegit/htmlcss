JavaScript란 무엇인가요?

JavaScript = 웹의 동적 프로그래밍 언어

동적(Dynamic)

웹페이지를 실시간으로 바꾸고, 사용자와 상호작용할 수 있게 함

예: 버튼을 클릭하면 내용이 바뀌거나, 입력창에 따라 결과가 실시간으로 변하는 기능
프로그래밍 언어

명령어(코드)로 컴퓨터에게 동작을 지시

예: 계산, 조건판단, 반복, 데이터 저장 등 다양한 논리를 구현
웹 표준

모든 브라우저에서 동작하는 공식 언어

HTML, CSS와 함께 웹의 3대 핵심 기술

실제 예시로 이해하기

JavaScript 없는 웹페이지:

정적인 페이지

수량: 1 (변경 불가)

총 가격: 10,000원 (고정)
JavaScript 적용 후

수량: 1

총 가격: 10,000원

인사이트: JavaScript가 있으면 웹페이지가 살아 움직이고, 사용자의 입력에 따라 즉시 반응합니다!

쉬운 비유로 이해하기

웹페이지를 사람이라고 생각해보세요:

  • HTML = 뼈대(골격)
  • CSS = 외모(피부, 옷, 스타일)
  • JavaScript = 뇌와 근육(생각하고 움직임)

왜 JavaScript를 배워야 할까요?

모든 웹사이트의 핵심

구글, 네이버, 유튜브 등 모든 사이트가 JavaScript로 동작

즉시 결과 확인

코드를 작성하면 바로 브라우저에서 결과를 볼 수 있음

취업과 실무 필수

웹 개발자, 프론트엔드, 백엔드, 앱 개발까지 필수 기술

배우기 쉬움

프로그래밍 입문자도 쉽게 시작 가능
