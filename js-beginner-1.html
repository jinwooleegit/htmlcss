JavaScript 초급편 1부

웹디자이너가 알아야 할 JavaScript 기초 개념과 활용법

웹디자이너를 위한 JS → 디자이너-개발자 소통 → 실무 JavaScript 효과

웹디자이너가 알아야 할 JavaScript란?

JavaScript = 디자인을 살아있게 만드는 마법

인터랙션 디자인의 핵심

사용자가 클릭, 스크롤, 호버할 때 반응하는 모든 효과

예: 버튼 호버 효과, 메뉴 애니메이션, 스크롤 시 나타나는 요소들
사용자 경험(UX)의 핵심

정적인 디자인을 동적이고 매력적인 경험으로 변환

예: 폼 검증, 로딩 애니메이션, 실시간 피드백
디자이너의 필수 지식

현대 웹디자인에서 JavaScript 이해는 선택이 아닌 필수

개발자와 소통하고, 실현 가능한 디자인을 만들기 위해 필요

디자이너 관점에서 이해하기

정적 디자인 vs 인터랙티브 디자인

정적 디자인 (HTML + CSS만)

메뉴가 항상 보임

이미지가 고정되어 있음
사용자 피드백 없음
인터랙티브 디자인 (+ JavaScript)

메뉴가 필요할 때만 나타남

클릭하면 커지는 요소
✓ 실시간 사용자 피드백

디자이너 인사이트: JavaScript로 디자인이 살아나고, 사용자와 대화하는 인터페이스가 됩니다!

디자이너를 위한 쉬운 비유

웹사이트를 매장 디자인이라고 생각해보세요:

  • HTML = 매장의 구조 (진열대, 계산대, 통로)
  • CSS = 매장의 인테리어 (조명, 색상, 레이아웃)
  • JavaScript = 매장의 인터랙션 (자동문, 터치스크린, 음성안내)
💡 디자이너 팁: 아무리 예쁜 매장(CSS)이라도 고객과 소통(JavaScript)하지 못하면 매력적이지 않아요!

웹디자이너가 JavaScript를 배워야 하는 이유

디자인 의도 실현

상상한 인터랙션과 애니메이션을 실제로 구현할 수 있음

개발자와의 소통

기술적 제약을 이해하고 현실적인 디자인 제안 가능

프로토타이핑 능력

정적 목업이 아닌 실제 동작하는 프로토타입 제작

커리어 확장

UI/UX 디자이너에서 프론트엔드 개발자로 성장 가능

디자이너가 알아야 할 JavaScript 기본 문법

디자이너 관점에서 보는 JavaScript 코드

// 이것은 주석입니다
console.log("안녕하세요!");
let message = "JavaScript 배우기";
alert(message);
주석: // 뒤의 내용은 실행되지 않음
명령문: console.log(), alert() 등
변수: let message = "값"
세미콜론: 문장의 끝을 표시 (선택사항)

디자이너가 알아야 할 JavaScript 연결 방법

CSS 연결처럼 JavaScript도 3가지 방법으로 연결할 수 있습니다

1

인라인 JavaScript

HTML 태그 내부 
<button onclick="alert('버튼 클릭!')">클릭하세요</button>
장점: 간단하고 빠름
단점: 코드가 길어지면 관리가 어려움
2

내부 JavaScript

HTML 파일 내부 
<script>
    console.log("내부 JavaScript 실행!");
    alert("Hello World!");
</script>
장점: HTML과 함께 관리 가능
단점: 재사용이 어려움
3

외부 JavaScript

추천 
<script src="script.js"></script>
장점: 재사용 가능, 관리 쉬움, 성능 좋음
단점: 파일이 하나 더 필요

실습 3: HTML에 JavaScript 추가하기

1

HTML 파일 만들기

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>JavaScript 실습</title>
</head>
<body>
    <h1>JavaScript 실습 페이지</h1>
    <button onclick="sayHello()">인사하기</button>
    
    <script>
        function sayHello() {
            alert("안녕하세요!");
        }
    </script>
</body>
</html>

위 코드를 메모장에 복사하고 test.html로 저장하세요!

2

브라우저에서 열기

저장한 HTML 파일을 더블클릭하여 브라우저에서 열어보세요!

3

버튼 클릭해보기

"인사하기" 버튼을 클릭하면 팝업이 나타납니다!

자주하는 실수와 해결법

❌ 실수 1

대소문자 구분 안하기

❌ 잘못된 코드
console.Log("Hello"); // Log는 대문자 L
✅ 올바른 코드
console.log("Hello"); // log는 소문자 l

해결법: JavaScript는 대소문자를 구분하므로 정확히 입력해야 합니다.

❌ 실수 2

따옴표 닫지 않기

❌ 잘못된 코드
alert("안녕하세요); // 따옴표 짝이 안 맞음
✅ 올바른 코드
alert("안녕하세요"); // 따옴표 짝이 맞음

해결법: 따옴표를 열었으면 반드시 같은 따옴표로 닫아야 합니다.

❌ 실수 3

함수 이름 오타내기

❌ 잘못된 코드
alret("Hello"); // alert 오타
✅ 올바른 코드
alert("Hello"); // alert 정확

해결법: 함수 이름을 정확히 외우고, 자동완성 기능을 활용하세요.
